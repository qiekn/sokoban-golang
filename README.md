## Control

| keys    | function                        |
| ------  | ------                          |
| w/s/a/d | move player                     |
| p       | toggle in-game pause menu       |
| r       | reset current level             |
| j       | next level            |
| k       | previous level            |
| q       | quit game (while in game-scene) |
